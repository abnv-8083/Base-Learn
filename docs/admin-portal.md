# 🛡️ Admin Portal — Base Learn LMS

## Overview
The Admin has **full control** of the entire Base Learn platform. This portal serves as the central command center for managing users, content, batches, and platform analytics.

---

## Access & Login
- Dedicated **Admin Login** page (separate from Student/Faculty/Instructor login)
- Secure authenticated session

---

## Dashboard
A central dashboard providing a real-time overview of the entire platform including:
- Total number of registered students
- Active batches and subjects
- Recent platform activity
- Payment and enrollment summary

---

## User Management

### Students
- Create student accounts
- Edit student account details
- Delete student accounts

### Faculty
- Create faculty accounts
- Edit faculty account details
- Delete faculty accounts

### Instructors
- Create instructor accounts

---

## Batch & Subject Management
- Manage batches:
  - Class 8
  - Class 9
  - Class 10
- Manage subjects under each class/batch

---

## Monitoring & Analytics

### Student Monitoring
- Student attendance records
- Lecture watch duration per student
- Lecture completion rates
- Course progress tracking

### Financial Monitoring
- Payment records
- Course enrollment payments

### Platform Activity
- Overall platform usage and engagement
- Lecture engagement metrics

---

## Analytics Dashboard
| Metric | Description |
|---|---|
| Total Students | Count of all registered students |
| Lecture Engagement | Views, watch time, completion rates |
| Attendance Reports | Per student, per batch, per subject |
| Payment Reports | Enrollment payments and history |
| Course Progress | Student progress across subjects/chapters |

---

## Technical Notes
- Mobile responsive interface required
- Secure login with role-based access control
- Notification system access
- Fast data loading for large datasets
