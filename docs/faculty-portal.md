# 👩‍🏫 Faculty Portal — Base Learn LMS

## Overview
Faculty members are responsible for conducting live classes and managing academic content such as notes and PYQ materials. They also have visibility into student learning activity and attendance.

---

## Access & Login
- Dedicated **Faculty Login** page (separate from Student/Admin/Instructor login)
- Secure authenticated session with Faculty-specific permissions

---

## Dashboard
Faculty dashboard provides an overview of:
- Assigned batches and subjects
- Upcoming scheduled live classes
- Student activity summary

---

## Live Classes

### Conducting Live Classes
- Host and conduct live streaming sessions
- Faculty has **host controls** during live class
- Students can see Faculty video; Faculty can see all Students
- Scheduled class timetable visible to Faculty

### Live Class Features
- Live streaming capability
- Student attendance tracked automatically during live sessions
- Host controls (mute, manage participants, etc.)

---

## Content Management

### Notes Upload
- Upload recorded class notes
- Upload live class notes
- Upload Previous Year Question (PYQ) materials

### Content Organization
Notes and materials are organized by:
- Class (Batch)
- Subject
- Chapter

---

## Student Monitoring

Faculty can track the following student metrics:
| Metric | Description |
|---|---|
| Student Attendance | Attendance per live class session |
| Lecture Watch Duration | How long students watch recorded lectures |
| Student Learning Activity | Overall engagement and activity on the platform |

---

## Technical Notes
- Mobile responsive interface
- Secure role-based access (Faculty cannot access Admin or Instructor controls)
- Notifications for scheduled class reminders
- Real-time attendance tracking during live sessions
